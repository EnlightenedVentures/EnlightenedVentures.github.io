
# Enlightened Ventures LLC — UmpClicker Site

Minimal static site designed for Apple App Store compliance (Support URL, Privacy Policy URL, Terms, Contact).

## Files to upload to your GitHub Pages repo (root)
- `index.html`
- `support.html`
- `privacy.html`
- `terms.html`
- `404.html`
- `robots.txt`
- `sitemap.txt`
- `favicon.ico` (replace with your icon)
- `assets/style.css`

## Notes
- Ensure GitHub Pages (Settings → Pages) publishes from `main` branch, root folder.
- Custom domain: `enlightenedventures.icu` with HTTPS enforced.
- Update the "View on the App Store" link on the home page once your App Store URL is available.
